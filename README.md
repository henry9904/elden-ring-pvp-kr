# ⚔️ Elden Ring PvP — Korean Translation & Guide Project

> 엘든링 PvP 룰셋 한국어 번역 및 가이드 프로젝트

---

## 📋 About / 소개

This repository contains a reformatted English version and Korean translations of the official Elden Ring PvP competitive ruleset, duel restrictions, and community guides.

이 레포지토리는 엘든링 PvP 경쟁 룰셋의 영문 정리본, 한국어 번역본, 결투 제한 규정, 그리고 커뮤니티 가이드를 포함한다.
