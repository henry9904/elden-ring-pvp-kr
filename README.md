## 📋 About / 소개

This repository contains a reformatted English version and a Korean translation of the official Elden Ring PvP competitive ruleset. Our goal is to improve the readability of the rules for the community.

이 레포지토리는 엘든링 PvP 경쟁 룰셋의 영문 정리본 및 한국어 번역본을 포함합니다. 커뮤니티 유저들이 복잡한 룰셋을 더 직관적으로 파악할 수 있도록 가시성을 개선하는 것을 목표로 합니다.
