# ⚔️ Elden Ring PvP Ruleset — Korean Translation Project

> 엘든링 PvP 룰셋 한국어 번역 및 리모델링 프로젝트

---

## 📋 About / 소개

This repository contains a reformatted English version, a Korean translation of the official Elden Ring PvP competitive ruleset, and a Korean translation of the Duel Restrictions document.

이 레포지토리는 엘든링 PvP 경쟁 룰셋의 영문 정리본, 한국어 번역본, 그리고 결투 제한 규정 한국어 번역본을 포함한다.
