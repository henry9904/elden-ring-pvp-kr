# ⚔️ Elden Ring PvP — Korean Translation & Guide Project

> 엘든링 PvP 룰셋 한국어 번역 및 가이드 프로젝트

---

## 📋 About / 소개

This repository contains Korean translations and redesigns of the official Elden Ring PvP ruleset, duel restrictions, and community guides.

이 레포지토리는 엘든링 PvP 결투 제한 규정, 룰셋, 커뮤니티 가이드의 한국어 번역 및 리디자인 문서를 포함한다.

> [!NOTE]
> **Standard DEN Rules** — 일반 결투 / 래더에서 사용하는 기본 규정이다.
> **Tournament Ruleset** — 토너먼트 전용 규정으로, 3회 전회 제한 등 추가 규칙이 포함된다. 토너먼트에 참가하는 경우에만 참고하면 된다.
