# ⚔️ Elden Ring PvP — Korean Translation & Guide Project

> 엘든링 PvP 룰셋 한국어 번역 및 가이드 프로젝트

A community-driven project providing Korean translations and redesigns of the official Elden Ring PvP ruleset, duel restrictions, and player guides. All documents are available in both English and Korean.

엘든링 PvP 결투 제한 규정, 룰셋, 커뮤니티 가이드의 한국어 번역 및 리디자인 프로젝트입니다.

---

> [!NOTE]
> **Standard DEN Rules** are the primary ruleset used for everyday duels and ladder play.
> **Tournament Ruleset** is supplementary — only relevant if you are participating in an organised tournament (includes additional rules such as the 3 Ash usage limit).
>
> **Standard DEN Rules**은 일반 결투 및 래더에서 사용하는 기본 규정입니다.
> **Tournament Ruleset**은 토너먼트 참가자를 위한 부록 규정입니다 (전회 3회 제한 등 추가 규칙 포함).
