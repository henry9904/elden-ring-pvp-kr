# ⚔️ Elden Ring PvP — Korean Translation & Guide Project

> 엘든링 PvP 한국어 번역 및 공략 프로젝트

---

## 📋 About / 소개

This repository contains Korean translations of the official Elden Ring PvP ruleset, duel restrictions, and community-contributed technique guides.

이 레포지토리는 엘든링 PvP 경쟁 룰셋 한국어 번역본과 커뮤니티 공략 가이드를 포함한다.
