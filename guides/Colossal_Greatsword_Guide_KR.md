# 🗡️ 엘든링 PvP — 특대검 (거대 그레이트소드) 통합 가이드

> [!NOTE]
> 영문 커뮤니티에서 **CGS = Curved Greatsword (곡선 대검)**을 의미한다.
> 기본 이해 및 특징 작성: **sin**
> 실전 운영 가이드 작성: **𝑄𝑅𝑄 [MAC]**
> 피드백 및 수정: **Darcy**
> 리디자인: **Crong**

> [!TIP]
> CR1, JR2, AoW 등 **용어가 낯설다면** → [📖 PvP Lexicon](../info/PvP_Lexicon_EN.md) 참고

---

## ⚡ 한줄 요약

> **특대검은 느리지만 강하다. 공격하는 게 아니라 상대를 유도해서 잡는 무기다.**

---

## 📊 장단점

| 장점 | 단점 |
|------|------|
| 높은 강인도 | 매우 느린 공격 속도 |
| 긴 사거리 | 긴 후딜레이 |
| 높은 강인도 데미지 | 제자리 공격 위주 |
| 엄청난 데미지 | 초근접 불리 |
| 맞딜에 강점 | 높은 레이턴시에서 불리 (단, 독 변질 사용 시 유효) |

> [!IMPORTANT]
> 공격을 받고도 공격이 끊기지 않는 강력한 하이퍼 아머가 특대검의 핵심 강점이다.
> 초보자 상대에 유리하나 **숙련자 상대 시 약점이 뚜렷하다.**

---

## 🎯 기본 운영 방식

| 단계 | 내용 |
|------|------|
| **압박** | 사정거리 안에서 언제 공격이 들어올지 모르는 상태를 만든다 |
| **혼란 유도** | 무빙 + 좌수 공격을 섞어 패닉 롤을 유도한다 |
| **롤캐치** | 유도된 패닉 롤을 찌르기 또는 강공 R2로 잡는다 |
| **사거리 의식** | 끝사거리를 항상 의식하라 — 상대 후딜캐치를 노릴 수 있다 |

> [!WARNING]
> **초근접은 피해라.** 압박 상황에서도 초근접 거리는 그소에게 불리하다.

---

## 🎮 공격 프레임 분석

| 공격 종류 | 시작 프레임 | 특징 |
|-----------|------------|------|
| **평타 (Standing R1)** | 17프레임 | Crouching R1을 스트레이핑으로 피하려는 상대에게 유효. 구르기로 피해도 뒤잡을 방지하며 강공 R2 세팅으로 이어짐 |
| **앉아 공격 (Crouching R1)** | 17프레임 | 빠르며 활용도가 가장 높음. **핵심 공격** |
| **강공 R2 (Heavy R2)** | — | **절대 빠뜨리면 안 되는 핵심 공격.** 평타와 조합 시 상대가 평타를 구르면 강공 R2 롤캐치에 취약해지는 딜레마를 만든다 |
| **구르기 공격 (Rolling R1)** | 15프레임 | 앉아 공격보다 2프레임 빠름. 보복 회피용 |
| **2~3타 체인** | 느림 | 거의 사용하지 않음. 명중 확실할 때만 사용 |
| **대시/점프 약공** | 20/18프레임 | 예측이 쉽고 후딜 김. 특정 상황에서 넓은 판정 활용 가능 |

> [!TIP]
> **평타 (Standing R1) + 강공 R2 조합이 핵심이다.** 상대가 평타를 구를 때마다 강공 R2 롤캐치에 더 취약해진다. 또한 평타를 상대의 구르기 방향과 반대로 꺽으면 뒤잡을 방지할 수 있다.

## 🔧 핵심 기술

### 1. 웨이브 대쉬

달리면서 앉았다 일어나기를 반복하면, 일어난 직후 **앉아있는 판정**이 남아 앉아 찌르기가 더 빠르게 나온다.

| 용도 | 설명 |
|------|------|
| **추적** | 상대와 거리가 벌어졌을 때 빠른 기동력으로 접근 |
| **찌르기 연계** | 앉기 판정을 이용한 빠른 찌르기 발동 |

> [!CAUTION]
> 웨이브 대쉬 중 **가드를 하면 앉기 상태가 풀린다.** 일어난 뒤 너무 늦게 공격하면 대시공이 나오니 주의.

---

### 2. 가드 캔슬 (Block Cancel)

가드 버튼을 짧게 눌러 **피벗 애니메이션(방향 전환 브레이크)을 방지**하고 회복 시간을 단축한다.

| 효과 | 설명 |
|------|------|
| 피벗 방지 | 90도 이상 회전 시 브레이크 방지 |
| 회복 단축 | 공격 후 회복 시간 감소 |

---

### 3. 꺽기 (방향 전환 공격)

락온을 풀고, 공격 도중 방향키를 움직여 히트박스를 더 빠르게 바꾸는 기술.

| 활용 | 효과 |
|------|------|
| 약공 꺽기 | 점프캐치 · 엇박캐치 가능 |
| 강공 꺽기 | 가능 |
| 대시공 꺽기 | 가능 |

> [!WARNING]
> 잘못 꺽으면 맞을 공격도 안 맞고 뒤잡각이 나온다. **경험이 필요하지만 고점이 높다.**

---

### 4. 드리머 스핀 (Dreamer Spins)

가드 캔슬과 앉기 캔슬을 결합한 스프린트 상태에서 방향키를 돌려 캐릭터를 회전시킨다.

| 효과 |
|------|
| 공격 타이밍 숨기기 |
| 상대 혼란 유발 |
| 패닉 롤 유도 |

> [!NOTE]
> 실전 활용은 제한적이지만 **애니메이션 블렌딩의 핵심 요소**다.

---

### 5. 헨디 테크

양잡을 풀거나 적용하는 소프트 스왑 시 **단시간 이속이 증가**한다.

| 상황 | 활용법 |
|------|--------|
| 상대가 멀리서 거리를 재고 있다 | 양잡을 풀고 대시강공 → 예상치 못한 거리에서 적중 |
| 상대가 아슬아슬하게 뒤로 달린다 | 찌르기·대시공으론 못 잡는 거리 → 양잡 풀고 대시강공 |
| 양잡 Crouching R1 적중 후 | 헨디 테크 → 상대가 반격하기 전에 오프핸드 공격 확정 |

---

## 🌀 애니메이션 블렌딩 & 믹스업

특대검의 공격은 모션이 길어 상대가 쉽게 예측할 수 있다. 블렌딩으로 공격 타이밍을 숨기는 것이 핵심이다.

| 블렌딩 기법 | 효과 |
|-------------|------|
| **앉기 공격 중 방향 조작** | 공격 시작 8프레임을 숨기고 9프레임만 반응 가능하게 만듦 |
| **앉기→평타 딜레이 공격** | Standing R1을 예상한 상대를 착각하게 만듦 |
| **평타 (Standing R1) → 강공 R2** | 평타를 구르면 강공 R2 롤캐치에 취약해지는 딜레마 생성. 상대를 위험한 선택의 기로에 세운다 |
| **언락 후 방향 조작** | 히트박스 방향 조작으로 숙련자 상대 연속 공격 가능 |
| **기름 도포** | 공격 애니메이션이 눈에 띄어 패닉 롤 유도 → 롤캐치 확률 증가 |

## ⚔️ 찌르기 후 심리전

### 찌른 후 상대 행동별 대응

| 상대 행동 | 대응 |
|-----------|------|
| 바로 공격한다 | 스핀 후 바로 찌르기 |
| 바로 구른다 | 찌르기 말고 롤캐치 |
| 점프한다 | 바로 찌르기 or 꺽어치기 |

### 롤캐치 대기 중 상대가 구르지 않는다면

| 상대 행동 | 대응 |
|-----------|------|
| 점프한다 | 암흑파 예측 또는 구르기 |
| 구르지 않는다 | 무빙으로 구르기 유도 후 압박 |
| 롤캐치 실패 | 구르기 공격 대비 or 거리 벌리고 재정비 |

## 📐 거리별 대응

### 거리가 벌어졌을 때

| 방법 | 설명 |
|------|------|
| 웨이브 대쉬 접근 | 기본 추적기 |
| 역점프 점공 | 점프 방향을 역으로 이용한 접근 |
| 대시 강공 리치 공격 | 긴 사거리 활용 |
| 대시공 꺽기 | 방향 전환으로 예측 회피 |

### 상대가 들어올 때

| 방법 | 설명 |
|------|------|
| 거리재기 후 찌르기 카운터 | 기본 대응 |
| 구르기 후 롤공 | *습관 들이지 말 것 — 점공 파훼 가능* |
| 상대가 점프로 들어올 때 | 대시강공 카운터 (예측 필요) |

### 상대가 뒤로 달릴 때

| 방법 | 설명 |
|------|------|
| 사거리 밖 공격 금지 | 카운터 맞음 |
| 헨디 테크 + 대시강공 | 이속 증가로 따라잡기 |

### 뒤잡 대응

> [!CAUTION]
> 그소는 **본인 기준 왼쪽이 약하다.** 상대가 왼쪽으로 붙어 뺑글 돌면 찌르기가 안 맞고 뒤잡이 나온다.
> → **좌수 무기로 견제하라.** 평타를 상대의 구르기 반대 방향으로 꺽는 것으로도 뒤잡을 방지할 수 있다.

## 🤚 좌수 활용

### 경직 높은 무기 (도끼, 데인의 발놀림)

| 활용 | 설명 |
|------|------|
| 견제 | 상대가 공격하려 할 때 경직 유발 |
| 딜교 | 좌수 경직 → 그소 찌르기 연계 |
| 트루콤 | 그소 찌르기 후 좌수 공격 확정 |
| Crouching R1 후 | 헨디 테크 → 상대 반격 전에 오프핸드 확정 |

### 경직 낮은 무기 (단검, 자검)

| 활용 | 설명 |
|------|------|
| 치고빠기 | 빠른 공격 후 빠른 구르기 |
| 딜교 | 치고빠기로 딜교 가능 |
| 트루콤 | 그소 찌르기 후 좌수 공격 확정 |

### 좌수 무기 선택 비교

| 무기 | 장점 | 단점 |
|------|------|------|
| **핸드엑스** | 강하고 경직 높음 | 공속 느리고 모션 큼 |
| **단검** | 빠르고 딜 나쁘지 않음 | 경직 없음 (활용 난이도 높음) |
| **자검** | 길고 아픔 | 무게 높고 경직 부족 |
| **데인의 발놀림** | 경직 높고 빠름, 모션 잘 안 보임 | 데미지·사거리 아쉬움 |

## 💎 탈리스만 세팅

### 기본 세팅

| 슬롯 | 기본 | 교체 옵션 |
|------|------|-----------|
| 1 | 체력 탈리스만 | 원휘석칼날 |
| 2 | 봉투방패 | 푸른 칠지인 |
| 3 | 무구 탈리스만 | — |
| 4 | 황나은 2 | — |

### sin 추천 세팅

- 창 탈리스만 (카운터 데미지 보너스)
- 산양 탈리스만 (강인도 33.3% 증가)
- 무구 탈리스만 (무게 부담 해결)
- 양손검 탈리스만 (양잡 시 공격력 증가)
- **되받아치는 교차수** (앉기 및 구르기 공격 +12%)

### Darcy의 탈리스만 조언

> [!TIP]
> **되받아치는 교차수 (Retaliatory Crossed-Tree)**는 특대검에 매우 효과적이다 — 앉기 공격과 구르기 공격 데미지를 12% 증가시킨다. 창 탈리스만보다 우선적으로 추천한다. 창 탈리스만은 이미 되받아치는 교차수를 착용 중일 때만 사용한다.

### 스왑 가이드

| 상황 | 선택 |
|------|------|
| 체력 탈리 빼고 봉투 한 번 더 쓰고 싶을 때 | 원휘석칼날로 스왑 |
| 더 많은 피해를 입는 상황 | 체력 탈리 → 원휘석칼날, 봉투방패 발동 |
| 쌍직 상대 시 *(실험 중)* | 산양탈리 착용 → 한손 공격 경직 없이 버팀 |

## 🔥 변질 옵션

| 변질 | 활용법 |
|------|--------|
| **냉기 변질** | 냉기그소로 동상 유발 후 중후 그소로 하드스왑 → 딜 고점 상승 |
| **독 변질** | 2~3타로 독 적용 후 안전 공격 가능. 독이 체력을 깎는 동안 상대는 공격할 수밖에 없는 상황이 만들어지며, 이는 맞딜에 강한 그소에게 유리하다 |

> [!TIP]
> 높은 레이턴시에서는 특대검 운영이 어렵지만, **독 변질을 사용하면 고레이턴시 환경에서도 효과적으로 운영할 수 있다.**

## 🩸 상성 분석

### 상대하기 어려운 무기

| 무기 | 이유 |
|------|------|
| **샴시르** | 속도가 압도적으로 빠름. 매우 어렵다 |
| **단검류** | 빠른 공격과 회피로 그소를 효과적으로 무력화 |
| **할버드 + 단검/도끼 조합** | 그소보다 긴 사거리, 유리한 근접전 |
| **자검 / 직검** | 빠른 회복과 강력한 공격 |
| **쌍직** | 그소의 약점인 점프 공격이 주 딜 수단 |

### 유리한 상성

> [!NOTE]
> 특대검은 **모든 헤비 무기군을 상대로 좋은 상성을 가진다.** 이길 수 없는 매치업은 없다 — 결국 실력의 문제다.

| **웨이브 대쉬 중 대시공이 나온다** | 가드하면 앉기 상태 해제됨 / 너무 늦게 공격해도 대시공 나옴 |
| **강공 R2를 쓰지 않는다** | 평타만으로는 불완전 — 항상 강공 R2를 후속 롤캐치 수단으로 준비해야 함 |
| **계속 뒤잡힌다** | 약공·강공 1타까지 / 초근접에서 대시공·대시강공 금지 / 너무 먼 점공 금지 / 평타를 구르기 반대 방향으로 꺽기 |
| **대시공으로 접근한다** | 맞지도 않는 대시공 쓰지 말고 찌르기를 위한 무빙으로 접근 |
| **무작정 공격한다** | 공격하는 게 아니라 상대의 행동을 유도하는 게 그소의 본질 |
| **고레이턴시에서 같은 방식으로 플레이** | 독 변질로 운영 방식 자체를 바꿀 것 |

---

## 🧠 심리전

특대검 플레이는 상대를 정신적으로 압박하고 행동을 유도하는 **마인드게임**에 가깝다.

- 평타 (Standing R1) + 강공 R2 조합은 상대에게 진짜 딜레마를 만든다 — 평타를 구르면 강공 R2에 취약해진다
- 상대가 공격을 기대하지 않는 순간을 노려 공격
- 움직임으로 상대를 혼란시키는 것이 중요
- 웨이브 대쉬, 스핀 등으로 패닉 롤을 유도하는 트릭을 적극 활용
- 버튼을 마구 누르기 시작하는 순간 이미 진 거다
