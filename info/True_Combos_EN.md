## 📖 Terminology

| Term | Description |
|------|-------------|
| **True Combo** | A sequence the opponent **cannot escape by rolling**. |
| **Pseudo Combo** | Not a true combo — the opponent can roll out — but the window is narrow enough that it catches most players in practice. |
| **Headshot** | A melee or projectile hit to the head that causes a long stagger, enabling otherwise impossible follow-ups. |
| **Free aim** | Manually unlocking the lock-on to redirect the hitbox of an attack. Direction (left, right, down) is noted where required. |
| **MH / OH** | Mainhand / Offhand weapon slot. |
