## 📖 Notation Guide

| Notation | Meaning |
|----------|---------|
| `RH` | Right Hand (Main-Hand) |
| `OH` | Off-Hand (Left Hand) |
| `PS` | Paired / Dual-Wield |
| `BH` | Both Hands |
| `R1 / R2` | Light / Heavy Attack |
| `L1` | Offhand Attack |
| `_1, _2...` | Attack chain number |
| `RUNNING_R` | Running attack |
| `LANDING_R` | Landing attack (from jump) |
| `BH_LAND` | Both-hand landing attack |
| `=>` | True combo into next attack |
| **Combo Tightness** | Input window for next attack (lower = stricter) |
