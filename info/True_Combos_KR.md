# 🔗 엘든링 PvP — 트루콤보 & 유사 콤보

> [!NOTE]
> 엘든링 PvP 근접 콤보 레퍼런스 자료입니다.
> 원문 데이터: **Hamster**. 한국어 마크다운 리디자인: **Crong**.

---

## 📖 용어 설명

| 용어 | 설명 |
|------|------|
| **트루콤보 (True Combo)** | 상대가 구르기로 **탈출할 수 없는** 연속 공격. |
| **유사 콤보 (Pseudo Combo)** | 트루콤보가 아니라 구르기로 탈출 가능하지만, 창이 매우 좁아 실전에서 대부분 잡힌다. |
| **헤드샷 (Headshot)** | 머리에 맞는 근접 또는 투사체 공격. 긴 경직을 유발해 일반적으로 불가능한 후속기를 가능하게 한다. |
| **자유 조준 (Free aim)** | 록온을 해제해 공격 히트박스를 수동으로 방향 조절. 필요한 경우 방향(왼쪽, 오른쪽, 아래)이 명시된다. |
| **메인핸드 / 오프핸드** | 주무기 / 보조 무기 슬롯. |
