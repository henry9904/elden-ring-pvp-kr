## 💨 전회 (AoW) — 3회 제한 규칙

> [!IMPORTANT]
> 모든 전회는 **사용 가능**하다. 단, **라운드당 전회 사용 횟수는 총 3회로 제한**된다.

- 라운드 내 어떤 전회든 **3회 초과 사용 시 라운드 패배** 처리
- 전회 **시전 중 끊겨도** 사용 횟수로 카운트된다
- 전회의 **후속 동작은 추가 카운트 없음** *(예: 끊기지 않은 연속 폭풍의 칼날 입력, 염격 후속 타격 등)*

| 예시 | 카운트 |
|------|:------:|
| 염격 L2 1회 사용 | 1 |
| 염격 R2 후속 타격 | 0 *(후속 동작)* |
| 폭풍의 칼날 3회 연속 입력 (끊기지 않음) | 1 *(1회 사용)* |
| 시전 중 끊긴 경우 | 1 |
| 어떤 전회든 4번째 사용 | ❌ 라운드 패배 |

**카운트 책임:**
- **토너먼트** — 양쪽 플레이어와 TO 모두 카운트. TO의 판정이 최종
- **래더** — 녹화를 권장. 양쪽 모두 녹화가 없으면 분쟁은 무시된다

## 📋 일반 판정

> [!NOTE]
> 토너먼트 진행 중 새로운 버그가 발견될 경우, TO(토너먼트 운영자)는 경기를 리셋할 권한을 가진다. 경기 전에 반드시 TO와 소통하라.

| 규정 |
|------|
| 규칙 위반 또는 메시지 힐 사용 시 라운드 패배 처리 |
| 금지된 장비를 장착하는 행위는 의도된 사용으로 간주한다. *(금지된 전회를 장착만 하고 사용하지 않는 경우는 예외)* |
| 상대방의 허가 없이 실수로 사용한 경우도 의도된 사용으로 간주한다 |
| 전체 경기 종료 후 빌드 공개 요청은 거부할 수 없다 |
| 기름 버프는 해당 기름을 금지하는 세팅으로 교체하기 전에 완전히 사라져야 한다 |
| 크로스보우, 활, 전회를 미리 준비하는 행위는 프리버프로 간주한다 |
| 상대방과 같은 월드에 진입한 이후 빌드 교체는 금지된다 |

> [!TIP]
> **경기 녹화를 강력히 권장한다.**
